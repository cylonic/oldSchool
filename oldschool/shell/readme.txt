HELP FILE
***************************************************

cd <directory>
Type cd, space, then the desired directory path to change to that directory.

clr
Type clr to clear the screen.

dir <directory>
Type dir, space, then the desired directroy to display the contents of the specified directory, or simply enter dir to display the contents of the current directory.

echo <comment>
Type echo, space, then a desired comment to display the desired comment on screen.

pause
Type pause to pause the shell, the shell will not accept commands until a single ENTER is pressed.

quit
Type quit to exit the shell.
